# Menu-de-jogos-em-C

## Descrição

Este projeto consiste em uma coleção de três jogos desenvolvidos em linguagem C para execução no terminal. O objetivo é aplicar conceitos fundamentais de programação, como estruturas de repetição, estruturas condicionais, vetores, matrizes, funções e manipulação de caracteres.

Os jogos disponíveis são:

- Jogo da Velha
- Jogo da Forca
- Batalha Naval

O usuário escolhe qual jogo deseja executar por meio de um menu principal.

---

## Funcionalidades

### Jogo da Velha

- Partida para dois jogadores.
- Validação das jogadas.
- Verificação automática de vitória.
- Verificação de empate.
- Exibição do tabuleiro após cada jogada.

### Jogo da Forca

- Palavra pré-definida.
- Controle do número de tentativas.
- Exibição das letras descobertas.
- Contagem de erros.
- Verificação de vitória ou derrota.

### Batalha Naval

- Posicionamento aleatório dos navios.
- Escolha das coordenadas pelo jogador.
- Indicação de acertos e erros.
- Contagem dos navios restantes.
